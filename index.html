🔑 Configura la chiave API

Per consultare le fonti meteo in tempo reale, questa app usa l'AI di Anthropic.
Ottieni una chiave gratuita su console.anthropic.com → sezione API Keys → Create Key.
La chiave viene salvata solo sul tuo dispositivo e non viene condivisa.

🌲
Inserisci la tua chiave API qui sopra, poi cerca qualsiasi località.
L'AI consulterà iLMeteo, 3bMeteo e meteo.it.
